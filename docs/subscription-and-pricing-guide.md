## How the Pricing Ladder Works

```
ONE-OFF CLIENT  -->  Standalone Report (£245-£995)
                     "I need one report for one site"

PROJECT CLIENT  -->  Bundle (£995-£6,995)
                     "I need several reports for one project"

REPEAT CLIENT   -->  Subscription (£399-£3,995/mo)
                     "I need reports every month"
```

Each level is cheaper per report than the one above it. Bundles beat standalone. Subscriptions beat bundles.
