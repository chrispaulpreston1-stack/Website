### Key Process Improvements

**A. Data Reuse Across Reports**
If a customer orders SFR then later orders FRA for the same site, the flood data already pulled by the SFR agents should be reused — no redundant API calls, faster turnaround, lower cost.

**B. Smart Upsell / Roadmap in Delivery**
Every report delivery should include a "What's Next?" section based on what the report found:
- SFR finds heritage constraints → recommend HIA
- SFR finds flood zone 2/3 → recommend FRA
- SFR finds TPOs → recommend Tree Survey
- FRA completed → recommend GDS for foundation strategy
- Planning permission granted → recommend Construction Readiness Pack

This is the **automated version of the Project Roadmap** — personalised to their site.

**C. Bundle Discount for Returning Customers**
If a customer already purchased SFR individually and then wants the Full Planning Suite, credit the SFR price against the bundle. This removes the penalty for not knowing about bundles upfront.

**D. Production Queue Prioritisation**
- Bundles get priority (higher revenue per order)
- Returning customers get priority (lower marginal cost due to data reuse)
- Subscription customers get priority
